# 🎮 Aprende Jugando - Tutorial de Programación

¡Bienvenido a este tutorial de programación con Python y Pygame!

## 📋 Objetivo
Crear un juego de preguntas de finanzas para secundaria, donde los alumnos aprenden programación mientras desarrollan un juego educativo.

## 📁 Archivos del Tutorial

| Archivo | Descripción | Nivel de dificultad |
|---------|-------------|-------------------|
| `settings.py` | Constantes (colores, posiciones) | 🟢 Básico |
| `questions.py` | Base de preguntas | 🟢 Básico |
| `main.py` | Punto de entrada | 🟢 Básico |
| `cannon.py` | Dibujo del cañón | 🟡 Medio |
| `blocks.py` | Lógica de bloques | 🟡 Medio |
| `ui.py` | Interfaz (HUD) | 🟡 Medio |
| `screens.py` | Pantallas inicio/fin | 🟡 Medio |
| `game.py` | Lógica principal | 🔴 Avanzado |

## 🚀 Cómo usar este tutorial

### Para el Tutor:
1. Copia toda la carpeta `tutorial/` a tus alumnos
2. Los archivos están numerados para seguir un orden
3. Cada archivo tiene tareas marcadas con "💡 TAREA"
4. Los huecos tienen "PISTA" para guiar a los alumnos

### Para el Alumno:
1. Abre los archivos en orden (del 1 al 8)
2. Lee los comentarios que dicen "COMPLETAR"
3. Usa las pistas para completar el código
4. Ejecuta `python main.py` para probar
5. Si tienes error, ¡no te rindas! Revisa la pista de nuevo

## 📦 Requisitos

```bash
pip install pygame
```

## 🕐 Orden recomendado

1. **settings.py** - Lee y entiende los colores/posiciones
2. **questions.py** - Agrega tus propias preguntas
3. **main.py** - Ejecuta y ve la pantalla negra
4. **cannon.py** - Completa el dibujo del cañón
5. **blocks.py** - Completa la lógica de bloques
6. **ui.py** - Personaliza la interfaz
7. **screens.py** - Modifica las pantallas
8. **game.py** - Completa la lógica del juego

## 🎯 Tareas por Archivo

### settings.py 🟢
- Cambiar colores (RGB)
- Cambiar resolución
- Modificar tiempo por pregunta

### questions.py 🟢
- Agregar más preguntas
- Cambiar las respuestas correctas

### main.py 🟢
- Cambiar tamaños de fuentes

### cannon.py 🟡
- Completar dibujo de base, ruedas y tubo
- Personalizar colores

### blocks.py 🟡
- Completar lógica de colisión
- Personalizar colores de bloques

### ui.py 🟡
- Mover elementos (cambiar coordenadas Y)
- Cambiar colores

### screens.py 🟡
- Cambiar textos (título, materia)
- Personalizar botones

### game.py 🔴
- Completar lógica de colisiones
- Entender el flujo del juego
